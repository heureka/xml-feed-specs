# Heureka Cpc

## Overview

**Element Name:** `HEUREKA_CPC`  
**Type:** decimal  
**Architecture:** Simple  
**Required:** No  

Heureka Cost Per Click bid amount. Must be a positive decimal number greater than 0.



## General Constraints

- **Optional field** - can be omitted
- **Minimum value:** 0.01

## XML Examples

```xml
<HEUREKA_CPC>5.50</HEUREKA_CPC>
```

```xml
<HEUREKA_CPC>12.00</HEUREKA_CPC>
```
