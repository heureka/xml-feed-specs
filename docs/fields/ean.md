# Ean

## Overview

**Element Name:** `EAN`<br>
**Type:** string<br>
**Architecture:** Simple<br>
**Required:** No<br>

European Article Number (EAN-13) barcode. Must be exactly 13 digits.


## General Constraints

- **Optional field** - can be omitted
- **Pattern:** `\d{13}`

## XML Examples

```xml
<EAN>1234567890123</EAN>
```
