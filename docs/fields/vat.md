# Vat

## Overview

**Element Name:** `VAT`<br>
**Type:** string<br>
**Architecture:** Simple<br>
**Required:** No<br>

VAT rate percentage applied to the product (e.g., "21" for 21% VAT).


## General Constraints

- **Optional field** - can be omitted

## XML Examples

```xml
<VAT>21</VAT>
```
